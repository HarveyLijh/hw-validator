üöÄ What is Chrome AI?

Chrome's built-in AI (Prompt API) allows web applications to access a local language model directly in the browser. This means:

Privacy First

All processing happens locally on your device. No data is sent to external servers.

Fast Response

No network latency - instant AI responses without internet dependency.

Free to Use

No API costs or usage limits - unlimited AI assistance for your assignments.

Works Offline

Once the model is downloaded, it works without an internet connection.

üîß Setup Requirements

Chrome AI is currently experimental

Follow these steps to enable Chrome's built-in AI:

1
Update Chrome
Ensure you're using Chrome version 127 or later. Check at chrome://settings/help
2
Enable Experimental Flags
Go to chrome://flags and enable:
  • Prompt API for Gemini Nano - Set to "Enabled"
  • Enables optimization guide on device - Set to "Enabled"
3
Restart Chrome
Close Chrome completely and restart it for the flags to take effect.
4
Model Download
The first time you use the AI, Chrome will download the language model (~1GB). This may take several minutes.
